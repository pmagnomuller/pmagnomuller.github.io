# Personal Website

My personal website and blog, built with Jekyll and powered by the [Minimal Mistakes](https://mmistakes.github.io/minimal-mistakes/) theme.

## Development

```bash
# Install dependencies
bundle install

# Start local server
bundle exec jekyll serve
```
